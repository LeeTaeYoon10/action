# 🎯 마케팅 액션 툴

다양한 광고 플랫폼의 성과 데이터를 통합 분석하고, 캠페인 액션을 관리하는 올인원 웹 기반 도구입니다.

## ✨ 주요 기능

### 📊 멀티 채널 통합 분석
- **메타(Meta)** - XLSX 파일 지원
- **구글(Google)** - CSV 파일 지원
- **구글 PMAX** - 독립적 캠페인 분석
- **틱톡(TikTok)** - XLSX 파일 지원
- **직접전환(Direct)** - 주문/전환 데이터 매핑

### 🎨 브랜드별 맞춤 테마
- 그래니 샐러드 (초록 테마)
- 베이스튠 (보라 테마)
- 브랜드별 독립적인 설정값 저장

### 📈 실시간 데이터 분석
- **정렬**: 테이블 헤더 클릭으로 오름/내림/초기순 정렬
- **검색**: 캠페인/광고명 실시간 필터
- **성과 기준값**: ROAS/CPA 기준값 커스터마이징
- **색상 코딩**: 위험/주의/양호 상태 시각화

### 💰 예산 관리 및 액션
- **증감액 자동 계산**: 매체별/전체 예산 변화 자동 계산 (원 단위)
- **액션 선택**: 유지/끄기/증액/감액/지켜보기
- **예산 변경**: 만원 단위 입력으로 즉시 증감액 반영
- **메모 기록**: 각 캠페인별 액션 메모

### 📸 리포트 내보내기
- **한 클릭 이미지 저장**: 분석 결과를 PNG로 다운로드
- **고화질**: 2배 배수 고화질 캡처
- **날짜 포함**: 자동으로 날짜가 포함된 파일명

### 💾 히스토리 관리
- **자동 저장**: 분석 후 자동으로 해당 날짜에 저장
- **기록 복원**: 이전 분석 결과 언제든 불러오기
- **기록 삭제**: 필요없는 기록 삭제 가능

## 🚀 사용 방법

### 1. 브랜드 선택
앱 시작 시 사용할 브랜드 선택 (그래니 샐러드 또는 베이스튠)

### 2. 데이터 업로드
각 광고 플랫폼의 파일을 드래그-드롭 또는 클릭으로 업로드:
- 메타: `.xlsx` 파일
- 구글: `.csv` 파일 (탭 구분)
- 구글 PMAX: `.csv` 파일
- 틱톡: `.xlsx` 파일
- 직접전환: `.csv` 파일

### 3. 날짜 설정
- **액션 날짜**: 이번 액션의 기준 날짜
- **보고 기간**: 분석 대상 기간 (시작~종료)

### 4. 분석 실행
"분석 시작" 버튼 클릭 → 데이터 파싱 및 렌더링

### 5. 데이터 조작
- **정렬**: 컬럼 헤더 클릭
- **검색**: 검색창에 캠페인명 입력
- **기준값 변경**: ⚙ 기준값 설정에서 ROAS/CPA 기준 조정

### 6. 액션 결정
각 캠페인/그룹별:
- **액션**: 드롭다운에서 선택 (유지/끄기/증액/감액/지켜보기)
- **예산**: 변경할 예산 입력 (만원 단위)
- **메모**: 액션 이유나 주의사항 기록

### 7. 결과 저장 및 공유
- **이미지 다운로드**: 분석 결과를 PNG로 저장
- **기록 보기**: 과거 분석 결과 확인

## 📊 성과 지표

### ROAS (광고 지출 수익률)
- 빨강(위험): 기준값 이하
- 노랑(주의): 기준값 ~ 주의 기준
- 초록(양호): 주의 기준 이상

### CPA (전환당 비용)
- 빨강(위험): 기준값 이상
- 노랑(주의): 양호 기준 ~ 기준값
- 초록(양호): 양호 기준 이하

### 직접 전환
- 직접전환 파일을 통해 매체 데이터와 실제 주문 연결
- 채널별 전용 키로 중복 합산 방지

## 🔧 기술 스택

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Data Parsing**: 
  - XLSX: SheetJS
  - CSV: PapaParse
- **Screenshot**: html2canvas
- **Storage**: localStorage (브랜드별 독립 저장)
- **Font**: Pretendard
